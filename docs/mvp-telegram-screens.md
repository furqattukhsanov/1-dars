# LolaMarket MVP — Telegram Mini App Ekranlar

## Barcha ekranlar

| # | Ekran | Izoh |
|---|-------|------|
| 1 | **Bosh sahifa** | Kategoriyalar, featured mahsulotlar, qidiruv paneli |
| 2 | **Katalog** | Filtr + saralash bilan mahsulotlar ro'yxati |
| 3 | **Mahsulot detail** | Foto, narx, o'lcham, material tafsilotlari + "Buyurtma" tugmasi |
| 4 | **Qidiruv** | Nom yoki kategoriya bo'yicha real-time qidiruv |
| 5 | **Savat** | Tanlangan mahsulotlar, miqdor, jami summa |
| 6 | **Buyurtma berish** | Yetkazib berish manzili, to'lov usuli, izoh |
| 7 | **Buyurtmalarim** | Faol va o'tgan buyurtmalar tarixi + holati |
| 8 | **Profil** | Kompaniya ma'lumotlari, manzil, bog'lanish |

## Minimal MVP (1-iteratsiya)

Faqat shu 5 ta ekran bilan ishga tushirish mumkin:

1. **Bosh sahifa** — kirish nuqtasi, kategoriyalar
2. **Katalog** — mahsulotlar ro'yxati
3. **Mahsulot detail** — to'liq ma'lumot va buyurtma tugmasi
4. **Savat** — tanlangan mahsulotlar
5. **Buyurtma berish** — oxirgi qadam

## Keyingi iteratsiya

- Qidiruv
- Buyurtmalarim
- Profil
