# Sprint 2 — Ma'lumotlar bazasi (Dars 9)

**Holat:** kutilmoqda

---

## Maqsad

LolaMarket uchun barcha ob'ektlarni ma'lumotlar bazasida yaratish va ular o'rtasidagi bog'lanishlarni o'rnatish.

---

## Bajariladigan vazifalar

- [ ] Supabase loyiha yaratish va ulanish
- [ ] Jadvallar yaratish:
  - `users` — ism, telefon, rol (xaridor / ishlab_chiqaruvchi / admin)
  - `sellers` — kompaniya, manzil, tasdiq holati, reyting
  - `products` — kategoriya, narx, rulon soni, rasm URL, seller ID
  - `orders` — xaridor ID, mahsulot ID, rulon miqdori, holat, BTS nuqta
  - `payments` — order ID, summa, komissiya, escrow holati
  - `disputes` — order ID, sabab, dalil URL, moderator qarori
  - `bts_points` — manzil, shahar, kod
- [ ] Jadvallar orasidagi foreign key bog'lanishlar
- [ ] Row Level Security (RLS): har rol faqat o'z ma'lumotini ko'radi
- [ ] Demo ma'lumotlar: 3 ta ishlab chiqaruvchi, 10 ta mahsulot
